## Bug Fixes

### Documentation Corrections

- **Removed `AllSetValues`** - Property handlers document `LastSetValue` only; `AllSetValues` was documented but never implemented
- **Fixed property customization examples** - Properties use `OnGet`/`OnSet` callbacks, not Get/Set prefix methods
- **Updated events support** - Events are now fully supported; removed from "defer to Moq" guidance
- **Fixed typo** - Corrected trailing space in generated-code.md example

### Files Updated

- `README.md` - Fixed property customization example
- `docs/guides/properties.md` - Removed `AllSetValues` reference
- `docs/reference/handler-api.md` - Removed `AllSetValues` from API reference
- `docs/reference/generated-code.md` - Fixed example code and typo
- `docs/knockoff-vs-moq.md` - Fixed property example, updated migration guidance
