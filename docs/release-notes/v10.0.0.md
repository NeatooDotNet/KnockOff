# v10.0.0 - Initial Release

**Release Date:** 2026-01-01
**NuGet:** [KnockOff 10.0.0](https://nuget.org/packages/KnockOff/10.0.0)

## Overview

Initial release of KnockOff, a Roslyn source generator for creating unit test stubs. Provides compile-time stub generation as a type-safe alternative to runtime mocking frameworks.

## Features

- **Source-generated stubs** - Compile-time code generation for interface implementations
- **Spy tracking** - Automatic call counting, argument capture, and invocation history
- **User-defined methods** - Define protected methods in partial class for stub behavior
- **Runtime callbacks** - OnCall/OnGet/OnSet for per-test customization
- **Properties** - Full support for get/set, get-only, and set-only properties
- **Methods** - Void methods, return values, single and multiple parameters
- **Async methods** - Task, Task<T>, ValueTask, ValueTask<T>
- **Generic interfaces** - Full generic type parameter support
- **Multiple interfaces** - Implement multiple interfaces in one stub
- **Interface inheritance** - Inherited interface members included
- **Indexers** - Get-only and get/set indexers with backing dictionaries
- **Events** - Event add/remove handlers
- **Named tuples** - Multi-parameter methods expose args as named tuples

## Target Frameworks

- .NET 8.0 (LTS)
- .NET 9.0 (STS)
- .NET 10.0 (LTS)

## Installation

```bash
dotnet add package KnockOff --version 10.0.0
```
