**Release Date:** 2026-01-02

## Overview

Documentation improvements and cleanup of stale references.

## What's Changed

### Documentation Fixes

- Fixed events support inconsistency in documentation
- Clarified default return behavior for non-nullable vs nullable types
- Added comprehensive out/ref parameter documentation
- Added method overloads documentation to migration guide

### Internal Cleanup

- Renamed `ExecutionInfo` property to `Spy` throughout codebase
- Cleaned up stale generated files with old naming

## Clarifications

### Default Return Values

| Scenario | Behavior |
|----------|----------|
| Nullable return type | Returns `null` |
| Non-nullable return type | Throws `InvalidOperationException` |
| Void methods | Completes silently |
| Task/ValueTask | Returns completed task with default value |
